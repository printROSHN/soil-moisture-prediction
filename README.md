# soil-moisture-prediction
EIOT PBL
Soil Moisture Prediction using Raspberry Pi

The purpose of this project is to alert the user when soil is dry. It also sends e-mail on user's device.

Working Video: https://www.youtube.com/watch?v=jWogLQ0COZc
